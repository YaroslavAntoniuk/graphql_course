# Before start
1. Create a MongoDB instance.
2. Create a `.env` file from example and pass your own data.
3. Change connection string in `app.js` if needed.

# Run backend
1. `npm i`
2. `npm run dev`

# Run frontend
1. `cd application`
2. `npm i`
3. `npm run start`
